Debug Actions
=============

Actions for debugging playbooks and capturing forensic evidence during execution.

Available Actions
-----------------

.. toctree::
   :maxdepth: 1

   pause
   screenshot

Overview
--------

Debug actions support forensic analysis and troubleshooting:

- **pause**: Interactive halt for manual inspection and verification
- **screenshot**: Capture VM screen state as visual evidence

These actions are essential for documenting forensic experiments and debugging playbook behavior.
