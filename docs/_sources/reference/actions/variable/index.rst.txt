Variable Actions
================

Actions for storing and managing variables during playbook execution.

Available Actions
-----------------

.. toctree::
   :maxdepth: 1

   save_timestamp
   save_variable

Overview
--------

Variable actions allow you to:

- Record timestamps for forensic timing analysis
- Store computed values from expressions
- Build dynamic values during execution
- Accumulate state across loop iterations

These variables can be used in subsequent actions, tests, or conditional flow control.
