Gui Automation Methods

find(image_name: str, minimal_similarity: float = 0.6, similarity_steps. float = 0.1) → list[guibot.match.Match] or None

Finds the image in the current screen and returns a list of matches or None if no match was found. Therefore, creates a chain of searches starting with a 90% similarity and decreasing the similarity by 10% until the image is found. If the image is not found with a similarity of the minimal similarity parameter the search is aborted and None is returned.

Parameters:

  • image_name (str) – the name of the image to find

  • minimal_similarity (float) – the minimal similarity to find the image with

  • similarity_steps (float) – the steps to decrease the similarity by

Returns:

a list of matches or None if no match was found

find_text(text: str) → list[guibot.match.Match] or None

Finds the text in the current screen and returns a list of matches or None if no match was found.

Parameters:

text (str) – the text to find

Returns:

a list of matches or None if no match was found

click(target_or_location: guibot.match.Match, modifiers: list[str] = [])

Left Click on the given match, previously found with the find method or manually created.

Parameters:

  • target_or_location – the match to click on (or an object with x and y coordinates)

  • modifiers – special keys to hold during clicking (e.g. [‘ctrl’, ‘shift’])

drag_and_drop(source: guibot.match.Match, target: guibot.match.Match, modifiers: list[str] = [])

Drags the source match to the target match.

Parameters:

  • source (guibot.match.Match) – the match to drag

  • target (guibot.match.Match) – the match to drop the source on

  • modifiers – special keys to hold during dragging (e.g. [‘ctrl’, ‘shift’])

press_keys(keys: list[str] or str)

Presses the given key or keys

Parameters:

keys (list[str] or str) – the keys to press

idle(timeout: int)

Idles the experiment for the given amount of time.

Parameters:

timeout (int) – the time to idle in seconds

exec_command(command: list, cwd: str = '')

Executes the given command in the shell.

Parameters:

  • command (list) – the command to execute

  • cwd (str) – the working directory to execute the command in

Test Methods

run_test(name: str)

Runs the test with the given name.

Parameters:

name (str) – the name of the test to run

run_tests(names: list[str])

Runs a list of tests with the given names.

Parameters:

names – the names of the tests to run

error(name: str, message: str)

Logs an error message with the given name and message.

Parameters:

  • name (str) – the name of the error

  • message (str) – the message of the error

Other Methods

In some cases it might be necessary to save certain data during the action phase that may be needed later on in the preformed tests (such as timestamps). Therefore the Experiment class offers the two methods described below.

save_variable(name: str, value: str)

Saves the given value under the given name in the variables file.

Parameters:

  • name (str) – the name to save the value under

  • value (str) – the value to save

save_time(timestamp_var_name: str)

Saves the current timestamp under the given name in the variables file.

Parameters:

timestamp_var_name (str) – the name to save the timestamp under
